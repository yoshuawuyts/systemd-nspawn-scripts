# systemd-nspawn-scripts
Various scripts to create images using systemd nspawn. Requires systemd and
a recent-ish kernel.

## Usage
```sh
$ ./sync          # Sync the repo to <remote>:~/scripts
$ ./build-alpine  # Build Alpine Linux
$ ./build-node    # Build Node.js
```
